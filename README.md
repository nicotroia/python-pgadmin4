# Installation

## Python Setup

```bash
brew install python3
```

Create a virtual env

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Install Packages

```bash
pip install -r requirements.txt
```

## Docker Setup

```bash
docker-compose up --build
```

## Running the Project

```bash
Visit PgAdmin 4 @ http://localhost:5050
```
